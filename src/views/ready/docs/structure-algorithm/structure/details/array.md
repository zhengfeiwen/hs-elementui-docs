
### 数组

##### 是可以再内存中连续存储多个元素的结构，在内存中的分配也是连续的，数组中的元素通过数组下标进行访问，数组下标从0开始。例如下面这段代码就是将数组的第一个元素赋值为 1。

    private data: Array<number> = new Array<number>()
    this.data[0] = 1
    console.log(this.data[0])

优点： 
1、按照索引查询元素速度快
2、按照索引遍历数组方便

缺点： 
1、数组的大小固定后就无法扩容了
2、数组只能存储一种类型的数据
3、添加，删除的操作慢，因为要移动其他的元素

适用场景： 
频繁查询，对存储空间要求不大，很少增加和删除的情况
